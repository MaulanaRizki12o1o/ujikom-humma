Web ini adalah halaman promosi dan prototype toko online alat tulis yang dibuat dengan PHP, HTML, CSS, dan JavaScript. File utamanya ada di toko_alattulis.php.

Isi utama web ini
Menampilkan navbar yang tetap berada di atas saat scroll.
Ada bagian hero/utama dengan slogan dan tombol “Lihat Koleksi”.
Menampilkan daftar produk unggulan seperti pena, pensil, buku sketsa, highlighter, dan lain-lain.
Ada bagian testimoni pelanggan.
Tersedia tombol masuk dan modal login sederhana.
Saat login berhasil, tampilan berubah dan pengguna bisa melihat tombol beli.
Terdapat fitur logout dan sesi login menggunakan PHP session.
Ciri khas
Data produk dan testimoni disimpan langsung di dalam array PHP, jadi ini lebih cocok sebagai landing page atau demo toko daripada toko online penuh.
Desainnya responsif dan terlihat modern
web ini terlihat simple tapi akan berguna
Fitur Utama
Menampilkan daftar produk alat tulis secara dinamis menggunakan array PHP.
Menampilkan testimoni pelanggan.
Sistem login sederhana menggunakan PHP Session.
Modal login interaktif tanpa berpindah halaman.
Navbar responsif dengan menu mobile.
Smooth scrolling pada navigasi.
Tampilan modern menggunakan HTML, CSS, dan JavaScript.
Logout pengguna.
Footer dengan informasi kontak perusahaan.
Teknologi yang Digunakan
PHP
HTML5
CSS3
JavaScript (Vanilla JS)
PHP Session
Struktur Data
Produk

Data produk disimpan dalam array PHP yang berisi:

Nama produk
Deskripsi produk
Harga
URL gambar

Data ini kemudian ditampilkan secara otomatis menggunakan perulangan (foreach).

Testimoni

Data testimoni pelanggan juga disimpan dalam array PHP dan ditampilkan secara dinamis pada halaman website.
